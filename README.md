## Built on one machine

The site, the dojo, the architecture behind it — built by **one person + AI, on one consumer PC.** No data center. No lab. You don't need permission or a fortune to build something real — just a machine, AI, and the will.

## How it's built

Plain, hand-written HTML & CSS. **Zero dependencies** — no frameworks, no external fonts, privacy-first (no cookies, no surveillance). Static, self-contained, served on GitHub Pages. The chess chamber inlines a real engine ([chess.js](https://github.com/jhlywa/chess.js), BSD-2). The simplest thing that could possibly work — built to last.

**Aesthetic:** gold on near-black, a single electric-blue charge, the SIGIL. Loving grace, not Big Brother.
